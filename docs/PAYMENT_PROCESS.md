# Bounty Payment Process

## Payment Methods
| Method | Min Amount | Processing Time |
|--------|-----------|----------------|
| GitHub Sponsors | $5 | 3-5 business days |
| PayPal | $10 | 1-3 business days |
| Crypto (USDC) | $25 | Same day |
| Bank Transfer | $50 | 5-7 business days |

## Payment Timeline
1. PR merged → Payment initiated
2. Verification (24-48h)
3. Payment sent
4. Confirmation email

## Previous Bounty Completions
If you have completed a bounty but haven't received payment:
1. Check that your wallet/payment info is in the PR description
2. Verify the PR was merged (not just closed)
3. Allow 7 business days for processing
4. If still unpaid, open a new issue with:
   - Link to the merged PR
   - Original bounty issue link
   - Payment method preference
   - Wallet/address for payment

## Disputes
For payment disputes, please provide:
- Transaction reference
- Expected amount
- Date of completion
- Any relevant communication
