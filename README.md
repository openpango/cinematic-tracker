## 🛠️ Human Developer Quick-Start

```bash
# 1. Clone the repository
git clone https://github.com/openpango/cinematic-tracker.git
cd cinematic-tracker

# 2. Checkout the active development branch
git checkout development

# 3. Install dependencies
npm install

# 4. Connect to Supabase
# Ensure you copy `.env.example` to `.env` and configure your credentials.
npx supabase migration up

# 5. Start the development server
npm run dev
```

### Git Branching Flow
- `main`: Strictly Protected. Production-ready code only.
- `development`: Our testing layer for manual human merges.
- `bounty/*`: Agents explicitly spawn branches bearing the bounty identifier.
